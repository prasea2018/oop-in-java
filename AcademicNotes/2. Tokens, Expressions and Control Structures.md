
# **UNIT: Data Types, Operators and Control Statements in Java**

## **1. Data Types in Java**

### **1.1 Meaning of Data Type**

A **data type** defines:

* The **type of value** a variable can store
* The **amount of memory** allocated
* The **range of values** allowed
* The **operations** that can be performed


Java is a **strongly typed language**, meaning all variables must be declared before use.

---

## **2. Primitive Data Types**

Java supports **8 primitive data types**, which are the most basic data types provided by the language.

---

## **2.1 Integer Data Types**

Integer data types store **whole numbers without decimal points**.

| Data Type | Size    | Range             |
| --------- | ------- | ----------------- |
| **byte**  | 1 byte  | -128 to 127       |
| **short** | 2 bytes | -32,768 to 32,767 |
| **int**   | 4 bytes | -2³¹ to 2³¹-1     |
| **long**  | 8 bytes | -2⁶³ to 2⁶³-1     |

### **Examples**

```java
int marks = 85;
byte age = 21;
long distance = 1500000L;
```

**Notes for Exams**

* `int` is the **default integer type**
* `long` values must end with `L`

---

1. Why is `int` preferred over `byte`?
2. Which integer data type uses the most memory?

---

## **2.2 Floating-Point Data Types**

Used to store **numbers with decimal values**.

| Data Type  | Size    | Precision       |
| ---------- | ------- | --------------- |
| **float**  | 4 bytes | Up to 7 digits  |
| **double** | 8 bytes | Up to 16 digits |

### **Examples**

```java
float interestRate = 7.5f;
double salary = 45250.75;
```

* `double` is the **default floating-point type**
* `float` must use suffix `f`

---


## **2.3 Character Data Type**

The **char** data type stores a **single character**.

```java
char grade = 'A';
char symbol = '#';
```

* Java uses **Unicode**, so `char` occupies **2 bytes**

---

## **2.4 Boolean Data Type**

Stores only two values:

* **true**
* **false**

```java
boolean isEligible = true;
```

Used mainly in **decision-making statements**

### **4.2 Constants**

### **4.3 Identifiers**

Identifiers are **names given to variables, methods, classes, or packages**.

#### **Rules**

* Must begin with a letter, `_`, or `$`
* Cannot start with a digit
* Cannot use Java keywords
* Case-sensitive

Valid:

```java
totalMarks, studentName
```

Invalid:

```java
int 2value;
```

---

### **4.4 Literals**

Literals are **fixed values** directly assigned to variables.

```java

char c = 'A';
boolean flag = true;
```

## **8. Arrays of Primitive Data Types**

An **array** stores multiple values of the **same data type**. Array index starts from **0**


### **Syntax 1: Declare + Allocate in separate line**

```java
int[] a;
a = new int[5];
```

#### Memory picture:

```
Heap memory:
Index:  0  1  2  3  4
Value:  0  0  0  0  0   (default int values)
```

* Array is created in **heap memory**
* Variable `a` holds a **reference** to the array
* `a.length` gives **5**

---

### **Syntax 2: Declare + Allocate in one line**

```java
int[] a = new int[5];
```

---

### **Syntax 3: Declare + initialize values**

```java
int[] a = {1, 2, 3, 4, 5};
```

#### Memory picture:

```
Index:  0  1  2  3  4
Value:  1  2  3  4  5
Length: 5
```

* You don't specify size — Java counts elements automatically.

## **12. Operators in Java**

Operators are symbols used to perform operations on variables and values. Using operators, we write **expressions**, and therefore it is very important to understand:

---

### **12.1 Arithmetic Operators**

Arithmetic operators perform basic mathematical operations.

| Operator | Meaning                   |
| -------- | ------------------------- |
| `+`      | Addition                  |
| `-`      | Subtraction               |
| `*`      | Multiplication            |
| `/`      | Division (gives quotient) |
| `%`      | Modulo (gives remainder)  |


#### **Area of a Triangle (Base & Height Formula)**

```java 

package expression;

import java.lang.*;
import java.util.*;


public class Expression 
{
  public static void main(String[] args) 
  {
    float base,height,area;
    System.out.println("Enter Base and Height");
    Scanner sc=new Scanner(System.in);
    base=sc.nextFloat();
    height=sc.nextFloat();
    area=base*height*0.5f;
    //area=1f/2f*base*height;
    //area=base*height/2;
    System.out.println("Area of Triangle is "+area);
  }
}
```

### **12.2 Relational Operators**

Relational operators are used to compare two values.
They always return a boolean value (true or false).

| Operator | Meaning                  |
| -------- | ------------------------ |
| `==`     | Equal to                 |
| `!=`     | Not equal to             |
| `>`      | Greater than             |
| `<`      | Less than                |
| `>=`     | Greater than or equal to |
| `<=`     | Less than or equal to    |


```java 
class RelationalOperatorDemo {
  public static void main(String[] args) {

    int a = 10;
    int b = 20;

    System.out.println(a == b);  // false
    System.out.println(a != b);  // true
    System.out.println(a > b);   // false
    System.out.println(a < b);   // true
    System.out.println(a >= b);  // false
    System.out.println(a <= b);  // true
  }
}

```
---

### **12.3 Logical Operators**

Logical operators are used to combine two or more conditions.
They always return a boolean value (true or false).

- Logical AND (&&)  
- Logical OR  (||)  
- Logical NOT (!)
```java 
class LogicalOperatorDemo {
  public static void main(String[] args) {

    int a = 10;
    int b = 20;

    System.out.println(a < b && a > 5);   // true
    System.out.println(a > b || a == 10); // true
    System.out.println(!(a > b));         // true
  }
}
```

---

### **12.4 Bitwise Operators**
Bitwise operators perform operations bit by bit on integer data types such as byte, short, int, long, and char

- Bitwise AND &  
- Bitwise OR  |  
- Bitwise XOR ^  
- Bitwise NOT ~
```java 
class BitwiseOperatorDemo {
  public static void main(String[] args) {
    int a = 5;   // 0101
    int b = 3;   // 0011

    System.out.println(a & b);  // 1
    System.out.println(a | b);  // 7
    System.out.println(a ^ b);  // 6
    System.out.println(~a);     // -6
  }
}
```

---

### **12.5 Assignment Operators**
Assignment operators are used to assign values to variables.
They can also perform an operation and assignment in a single step.

| Operator | Name                | Description               |
| -------- | ------------------- | ------------------------- |
| `=`      | Simple assignment   | Assigns value to variable |
| `+=`     | Add and assign      | `a += b` → `a = a + b`    |
| `-=`     | Subtract and assign | `a -= b` → `a = a - b`    |
| `*=`     | Multiply and assign | `a *= b` → `a = a * b`    |

```java 
class AssignmentOperatorDemo {
  public static void main(String[] args) {

    int a = 10;

    a += 5;   // a = a + 5
    System.out.println(a); // 15

    a -= 3;   // a = a - 3
    System.out.println(a); // 12

    a *= 2;   // a = a * 2
    System.out.println(a); // 24
  }
}
```

## **13. Control Statements**

Control statements control the **flow of program execution**.

---

## **13.1 Branching Statements**

### **if Statement**

```java
if (marks >= 40) {
    System.out.println("Pass");
}
```

---

### **switch Statement**

```java
switch(day) {
  case 1: System.out.println("Sunday");
    break;
}
```

`switch` is used when choices are fixed.

---

## **13.2 Looping Statements**

### **while Loop**

```java
while(i <= 5) {
    i++;
}
```

---

### **do-while Loop**

Executes **at least once**.

```java
do {
    i++;
} while(i <= 5);
```

---

### **for Loop**

```java
for(int i=1; i<=5; i++) {
  System.out.println(i);
}
```

---

## **13.3 Jumping Statements**

### **break**

Terminates loop or switch.

### **continue**

Skips current iteration.

### **return**

Returns control from method.
