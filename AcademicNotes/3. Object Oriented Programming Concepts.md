## **1. Fundamentals of Classes in Java**

### **1.1 Meaning of a Class**

A **class** is a **blueprint or template** used to create objects.
It defines:

* **Data members** (variables)
* **Member functions** (methods)

**Exam Definition:**

> A class is a user-defined data type that represents a group of objects having common properties and behaviors.

---

### **1.2 A Simple Class**

### **Syntax**

```java
class ClassName {
  // data members
  // methods
}
```

### **Example**

```java
class Student {
  int rollNo;
  String name;

  void display() {
      System.out.println(rollNo + " " + name);
  }
}
```

## **3. Adding Methods to a Class**

### **Meaning of Method**

A **method** is a block of code that performs a specific task.

Methods define the **behavior** of objects.

---

### **Syntax**

## **5. Abstraction**

### **Meaning of Abstraction**

**Abstraction** means **hiding implementation details** and showing only essential features.


To hide data, we use **`private` access modifier**
* Properties cannot be accessed directly outside the class.
* They can only be accessed **inside** the class.

```java
class Rectangle {
  private int length;
  private int breadth;

  public int area() { 
    return length * breadth;
  }
  public int perimeter() { 
    return 2 * (length + breadth);
  }
}
```

**5. Problem Without Data Hiding**

If `length` and `breadth` are public:

```java
public class Program {
  public static void main(String[] args) {
    Rectangle r = new Rectangle();
    r.length = -10;   // invalid
    r.breadth = 5;
  }
}
```

This leads to incorrect values causing invalid area/perimeter. Thus, data must be protected.

**How to Access Hidden Data? (Getters and Setters)**

To read and modify private data, we provide **property methods**:

**Getters (read)**

Return the value.

```java
public int getLength() {
  return length;
}
```

**Setters (write)**

Set the value â€” can include validation.

```java
public void setLength(int length) {
  if (length > 0)
    this.length = length;
  else
    this.length = 0; // validation
}
```

- Comple code demo of Data Hiding 
```java 

class Rectangle
{
	private double length;
	private double breadth;

	public double getLength()
	{
		return length;
	}

	public double getBreadth()

	public void setLength(double l)

}

public class Program {

	public static void main(String[] args) {
		Rectangle r=new Rectangle();

		System.out.println("Perimeter "+r.perimeter());

```


## **6. Encapsulation**

### **Meaning of Encapsulation**

**Encapsulation** is the process of **wrapping data and methods together** into a single unit (class).

Achieved using:

* Private variables
* Public getter and setter methods

---

### **Example**

```java
class Account {
    private double balance;

    public void setBalance(double b) {
        balance = b;
    }

    public double getBalance() {
        return balance;
    }
}
```

---

### **Advantages**

* Data security
* Controlled access
* Improved maintainability

## **7. Using the `this` Keyword**

### **Meaning**

The **`this` keyword** refers to the **current object**.

---

### **Uses of `this`**

2. Invoke current class methods
3. Call constructors

    int id;

## **11. Recursion**

### **Meaning**

A method calling **itself** is called recursion.

---

### **Example**

```java

    if(n == 1)
        return 1;
    return n * factorial(n - 1);
}
```

ðŸ“Œ Must have:

* Base condition
* Recursive call
